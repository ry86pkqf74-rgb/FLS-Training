# Scoring vs Coaching

The system must not represent AI-derived coaching analytics as official FLS certification scoring. Official FLS scoring uses normalized task scoring and certified procedures; this system provides AI-assisted training feedback unless explicitly validated and approved for certification use.

## Official/Rubric-Derived Fields

- `task_id`
- `max_score`
- completion time
- penalties
- final score
- critical errors
- confidence
- cannot determine

## Coaching Fields

- strengths
- weaknesses
- recommended drills
- next practice plan
- longitudinal progress

## Experimental Fields

- z-scores
- movement metrics
- cohort-relative analytics

Experimental fields are coaching aids only. They should appear in an appendix and must not be used as validated pass/fail evidence unless the cohort and metric definitions have been approved for that use.
