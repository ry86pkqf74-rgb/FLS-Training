# Deprecated Feedback v1 Generators

These modules are preserved for audit and historical training-data lineage only.

Use `src.reporting.report_v3.generate_report_v3` and
`src.reporting.render_markdown_v3.render_markdown_v3` for current report
generation. The archived generators are Task-5-centric and may contain legacy
language that v003 explicitly disallows, including hard-coded denominators and
unsupported readiness claims.
